# 🤖 AI Mock Interview App

A simple AI-based mock interview application that helps users practice interview questions and improve their answers with structured feedback.

---

## 🚀 Features
- AI-generated interview questions  
- Practice answer input system  
- Instant feedback system  
- Score tracking  
- Clean and simple UI  

---

## 🛠️ Tech Stack
- Flutter  
- Dart  

---

## 📱 Project Goal
To help students and job seekers improve their interview skills through practice and feedback.
